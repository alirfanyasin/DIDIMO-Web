# DIDIMO WEBSITE

## Instalation
Setelah di clone repositorynya silahkan tulis perintah dibawah ini
```sh
composer install
```
dan
```sh
npm install
```

Lalu copy file .env 
```sh
cp .env.example .env
```

Setelah itu lakukan generate key
```sh
php artisan key:generate
```

## Running Project
```sh
php artisan serve
```

jika ingin menjalan kan vitenya
```sh
npm run dev
```
